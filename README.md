# Fee Payment System

## Getting Started
1. Open CMD prompt as administrator
2. Navigate to Fee-Payment-System Directory
3. Run the following commands:
    1. set FLASK_APP=main.py
    2. set FLASK_DEBUG=1
4. Run python main.py
5. Open Web Browser
6. Goto 127.0.0.1:5000
